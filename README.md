# 勉強会用に作成した「お前は今日なにしてたんだよ。」です

## 主な機能
- ローカル通知が来る
- 返信できる
- 返信内容と時間が保存される
- 保存されたコンテンツが一覧表示される

各フォルダごとに以下の通りstep分けしてあります 

- 初期状態
- 通知許可を送る
- 通知を送る
- 返信をできるようにする
- 返信を保存するようにする
- core dataから一覧を表示する
